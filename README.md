# Personal Expense Tracker

## Overview
This is a web-based Expense Tracker application built using Python and Streamlit. It allows users to manage their daily expenses, view spending, and analyze data using charts.

## Features
- Add daily expenses
- View all expenses in table format
- Calculate total spending
- Category-wise bar chart visualization
- Pie chart for expense distribution
- Download expense data as CSV

## Tech Stack
- Python
- Pandas
- Streamlit
- Matplotlib

## **How to Run Locally**

Clone the repository:
git clone https://github.com/s0r3j6/expense_tracker

Navigate to project folder:
cd expense_tracker

Install dependencies:
pip install -r Requirements.txt

Run the app:
streamlit run app.py
